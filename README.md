# 📊 CallOps Analytics

Plataforma web para monitoramento e supervisão de indicadores operacionais em ambientes de Call Center.

O projeto tem como objetivo fornecer uma visão estratégica da performance da equipe, permitindo análise consolidada no dashboard e visualização individual de resultados por atendente.

---

## 🚀 Funcionalidades Implementadas

- ✅ Tela de Login com interface moderna
- ✅ Alternância entre Light Mode e Dark Mode
- ✅ Persistência de tema via localStorage
- ✅ Estrutura organizada em HTML, CSS e JavaScript
- ✅ Layout responsivo
- ✅ Footer profissional

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

---
## 🎯 Objetivo do Projeto

O CallOps Analytics foi concebido como base para uma aplicação completa de supervisão operacional, com foco em:

- Organização e escalabilidade de código
- Boas práticas de separação de responsabilidades
- Estrutura preparada para integração com API
- Futuro uso de banco de dados relacional (ex: PostgreSQL)
